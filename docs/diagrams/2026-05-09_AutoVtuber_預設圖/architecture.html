🛡️ Safety Layer 子圖 · 安全層
硬體守衛 1 秒輪詢 → threading.Event 通知所有 stage abort。RAM spike 期間用 ModelLoader 序列化避免 OOM。
flowchart TB
    subgraph HG["🛡️ HardwareGuard · 硬體守衛"]
        HG_POLL[1 秒輪詢
1s polling interval]
        HG_RAM[RAM 監控
warn 90% / abort 97%]
        HG_GPU[GPU 監控
VRAM + 溫度]
        HG_DISK[磁碟監控
output dir 剩餘空間]
        HG_HYS[Hysteresis 3s
避免 flicker]
        HG_EVT[(threading.Event
abort signal)]
        HG_POLL --> HG_RAM
        HG_POLL --> HG_GPU
        HG_POLL --> HG_DISK
        HG_RAM --> HG_HYS
        HG_GPU --> HG_HYS
        HG_DISK --> HG_HYS
        HG_HYS -- "trigger" --> HG_EVT
    end

    subgraph ML["⚖️ ModelLoader · 模型調度器"]
        ML_RULE[單一重模型於 GPU
Single heavy model rule]
        ML_SEQ[Sequence:
SDXL → unload
TripoSR → unload]
        ML_PURE[Pure GPU mode .to cuda
不用 sequential_cpu_offload]
        ML_RULE --> ML_SEQ
        ML_SEQ --> ML_PURE
    end
